# 동대문구 노인보호구역 위험도 평가

동대문구 노인보호구역의 교통사고 위험도를 분석하고 평가하는 프로젝트입니다.

## 프로젝트 개요

본 프로젝트는 동대문구 내 노인보호구역의 교통사고 위험도를 MaxEnt 모델을 활용하여 분석하고, 다양한 환경 변수와 시설 데이터를 종합하여 위험도를 평가합니다.

## 프로젝트 구조

```
동대문구_노인보호구역_위험도평가/
├── data/              # 원본 데이터 파일
│   ├── *.csv          # CSV 데이터 파일
│   └── *.shp          # Shapefile 데이터
├── notebooks/         # Jupyter Notebook 분석 파일
│   ├── 데이터수집.ipynb
│   ├── 동대문구 지도 격자.ipynb
│   └── 환경변수asc변환코드.ipynb
├── src/               # Python 스크립트
│   ├── grid_analysis.py          # 격자 분석 스크립트
│   └── 환경변수_시각화.py        # 환경변수 시각화 스크립트
├── models/            # MaxEnt 모델 관련 파일
│   └── 환경변수/      # 환경변수 ASC 파일
├── results/            # 분석 결과물
│   ├── *.html         # Folium 지도 시각화
│   └── *.png          # 분석 결과 이미지
└── docs/              # 문서 및 발표 자료
    ├── PPT 및 발표대본/
    └── 동대문구 분석 보고서.pdf
```

## 주요 기능

1. **데이터 수집 및 전처리**
   - 동대문구 내 다양한 시설 데이터 수집
   - 노인보호구역 위치 정보 수집
   - 교통사고 발생 데이터 수집

2. **환경변수 생성**
   - 시설 밀도 분석
   - 교통 인프라 분석
   - 보행 환경 분석

3. **위험도 평가**
   - MaxEnt 모델을 활용한 위험도 예측
   - 격자별 위험도 분석
   - 시설 접근성 분석

4. **시각화**
   - Folium을 활용한 인터랙티브 지도
   - 위험도 히트맵
   - 시설 분포 지도

## 설치 및 실행

### 요구사항

Python 3.8 이상

### 패키지 설치

```bash
pip install -r requirements.txt
```

### 실행 방법

1. 데이터 준비: `data/` 폴더에 필요한 데이터 파일 배치
2. 노트북 실행: `notebooks/` 폴더의 Jupyter Notebook 실행
3. 스크립트 실행: `src/` 폴더의 Python 스크립트 실행

## 주요 데이터

- 노인보호구역 위치 정보
- 교통사고 발생 데이터
- 시설 데이터 (가로등, CCTV, 횡단보도, 병원 등)
- 교통 인프라 데이터 (교차로, 버스정류소, 지하철역 등)

## 분석 결과

분석 결과는 `results/` 폴더에서 확인할 수 있습니다.
