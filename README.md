# 🚸 동대문구 노인보호구역 위험도 평가

동대문구 노인보호구역의 교통사고 위험도를 분석하고 평가하는 프로젝트입니다.

## 프로젝트 개요

본 프로젝트는 동대문구 내 노인보호구역의 교통사고 위험도를 **MaxEnt (Maximum Entropy) 모델**을 활용하여 분석하고, 다양한 환경 변수와 시설 데이터를 종합하여 위험도를 평가합니다.

### 연구 배경

고령화 사회로 진입하면서 노인 교통사고가 증가하고 있으며, 특히 노인보호구역 주변에서의 사고 위험도 평가가 중요해지고 있습니다. 본 연구는 동대문구를 대상으로 공간 분석 기법을 활용하여 노인보호구역의 교통사고 위험도를 정량적으로 평가하고, 위험 지역을 식별하여 정책적 개선 방안을 제시합니다.

### 연구 목적

1. 동대문구 노인보호구역 주변의 교통사고 위험도를 공간적으로 분석
2. 다양한 환경 변수(시설, 교통 인프라, 보행 환경)가 위험도에 미치는 영향 분석
3. MaxEnt 모델을 활용한 위험도 예측 및 시각화
4. 위험 지역 식별 및 정책 개선 방안 제시

## 연구 방법론

### 1. 데이터 수집 및 전처리

#### 수집 데이터
- **노인보호구역 정보**: 노인장애인보호구역 위치 및 현황 데이터
- **교통사고 데이터**: 보행 노인 사고 발생 위치 데이터
- **시설 데이터**:
  - 안전 시설: 가로등, CCTV, 보행등, LED 횡단보도, 횡단보도
  - 교통 인프라: 교차로(삼거리, 사거리, 단일로), 버스정류소, 지하철역, 차량출입구
  - 의료/복지 시설: 병의원, 노인여가복지시설, 노인일자리지원기관, 재가노인복지시설
  - 생활 인프라: 전통시장, 실버존
  - 교통 단속: 무인교통단속카메라
- **인구 데이터**: 노인 인구 현황 데이터

#### 전처리 과정
- 좌표계 변환 (EPSG:5179 → EPSG:4326, EPSG:5186)
- 데이터 인코딩 처리 (UTF-8, CP949, EUC-KR)
- 공간 데이터 통합 및 정제

### 2. 환경변수 생성

다양한 환경 변수를 ASC (ASCII Grid) 형식으로 변환하여 MaxEnt 모델 입력 데이터로 활용:

- **교통 인프라 변수**: 버스정류소 밀도, 교차로 밀도(삼거리/사거리/단일로), 차량출입구 밀도
- **안전 시설 변수**: 가로등 밀도, CCTV 밀도(차량/보행), 횡단보도 밀도, 보행등 밀도, LED 밀도
- **의료/복지 시설 변수**: 병원 밀도(로그 변환), 노인시설 밀도, 실버존 밀도
- **생활 인프라 변수**: 전통시장 KDE 밀도(로그 가중치 적용)
- **인구 변수**: 노인 인구 비율

### 3. MaxEnt 모델을 활용한 위험도 평가

- **MaxEnt 모델**: 종속 변수(교통사고 발생 위치)와 독립 변수(환경 변수) 간의 관계를 학습
- **격자 기반 분석**: 동대문구를 30×30 격자로 분할하여 각 격자별 위험도 계산
- **위험도 지수**: 0~1 사이의 값으로 표현 (높을수록 위험)
- **모델 검증**: AUC (Area Under Curve) 값을 통한 모델 성능 평가

### 4. 공간 분석 및 시각화

- **Folium 지도 시각화**: 인터랙티브 웹 지도를 통한 위험도 시각화
- **위험도 히트맵**: 격자별 위험도를 색상으로 표현
- **시설 분포 지도**: 각종 시설의 위치 및 밀도 시각화
- **노인보호구역 분석**: 보호구역 중심 300m 반경 내 위험도 분석

## 주요 분석 결과

### 위험도 분포 특성

- 동대문구 내 노인보호구역 주변의 위험도는 지역별로 상이하게 분포
- 교통량이 많고 시설이 밀집된 지역에서 상대적으로 높은 위험도 확인
- 노인 인구 밀도와 위험도 간의 상관관계 분석

### 주요 영향 변수

MaxEnt 모델 분석 결과, 위험도에 영향을 미치는 주요 변수:

1. **교통 인프라**: 교차로 밀도, 차량출입구 밀도
2. **안전 시설**: CCTV 밀도, 횡단보도 밀도, 가로등 밀도
3. **생활 인프라**: 전통시장 밀도, 노인시설 접근성
4. **인구 특성**: 노인 인구 비율

### 정책 제언

1. **고위험 지역 우선 개선**: 위험도가 높은 격자 지역에 대한 안전 시설 확충
2. **시설 배치 최적화**: 노인보호구역 주변 안전 시설의 공간적 배치 개선
3. **교통 환경 개선**: 교차로 및 차량출입구 주변 보행 환경 개선
4. **모니터링 체계 구축**: 지속적인 위험도 모니터링 및 평가 체계 구축

## 분석 결과물

분석 결과는 `results/` 폴더에서 확인할 수 있습니다:
